🔬 提示詞核心結構

每一個有效的 Nano Banana 2 提示詞都應涵蓋以下要素（不需全部，但越具體越好）

要素 1

Subject 主體

主體是誰或什麼

一位穿著精緻和服的女子 / 清晨時分的香港街市
要素 2

Action / State 動作

發生什麼事 / 狀態

正在街邊品嚐雞蛋仔 / 在雨中排隊等巴士
要素 3

Environment 環境

環境 / 場景 / 時間天氣

暴雨時段的旺角街頭，晚間七時 / 昭和年間的東京淺草寺
要素 4

Composition / Camera 構圖

構圖 / 鏡頭 / 角度

標準50mm鏡頭，人物居中，淺景深
要素 5

Lighting / Mood 光線

光線 / 情緒

黃金時段，金色逆光邊緣光 / 冷調實驗室白光
要素 6

Style 風格

藝術或攝影風格參考

電影感色調、柔和暖色光、復古菲林感
要素 7

Exclusions 排除

明確說明不要什麼

不要文字、不要浮水印、不要路人甲

📐 通用骨架公式

[主體] + [動作/狀態] + [場景環境] + [攝影/藝術風格] + [鏡頭/構圖] + [光線/色調] + [額外細節] + [排除項目]

✏️ 圖像編輯公式

保持（必須不變的部分）+ 改變（要修改的項目）+ 位置/強度 + 風格一致 + 光線匹配 + 排除副作用

💡 7 大關鍵技巧

掌握這些技巧，讓你的提示詞從「可用」提升至「專業」

1
🎬

像創意總監一樣說話

NB2 是思考模型，理解意圖而非匹配關鍵字。用自然語言描述你想要的畫面。

貓 夜晚 城市 霓虹燈
一隻淘氣的貓試圖從廚房料理台上偷魚，溫暖的廚房光線，鮮豔漫畫風格
2
📷

使用具體的攝影語言

提到真實相機、菲林、鏡頭型號，NB2 會精確模擬這些設備的視覺行為。

使用萊卡 M10 拍攝，50mm Summilux 鏡頭，Kodak Portra 400 菲林
85mm f/1.4 大光圈鏡頭，背景柔化成夢幻散景
3
📝

文字渲染記得用雙引號

想在圖像中顯示特定文字，必須用雙引號包圍，否則可能被當成構圖指令。

在圖上寫 灣仔茶餐廳
顯示文字：「灣仔茶餐廳」（雙引號 = 字面渲染）
4
🎨

加入參考風格，一鍵激活

一個好的參考詞可以一次激活多種視覺關聯，是最高效的提示詞捷徑。

王家衛電影配色，參考《花樣年華》的2000年風格
國家地理雜誌人文紀實風格
5
🚫

善用負面提示

清楚告訴模型「不要」什麼，通常比只說「要」什麼更有效。

只描述想要的元素
不要文字、不要浮水印、不要其他路人、不要模糊
6
🔄

迭代編輯是 NB2 強項

圖像 80% 正確時，不要全部重來，給出精確的修改指令即可。

很好，但請把光線改成夕陽效果，並把文字改成藍色
7
📐

每次都加上比例（AR）

資產類型 + 比例決定生成方向，不指定比例會使用預設，可能不符需求。

16:9 電影分鏡 / 4:5 資訊圖表 / 9:16 社群媒體直式

📋 10 個場景化模板提示詞

整合兩份文件精華，可直接複製修改使用

【模板一】攝影寫真風格

寫真 · 人物 · 城市
Prompt 結構
[主體描述，包括穿著/表情/姿勢]，拍攝於 [地點/時間/天氣]， [相機型號]，[鏡頭]，[構圖方式]，[景深]，[光線方向與色溫]， [風格參考]。不要出現 [排除元素]。
範例提示詞（可直接使用）

一位穿著淺色風衣的男子站在雨天的香港旺角街道上，濕潤的路面有霓虹燈倒影。寫實攝影風格，35mm鏡頭，中景，淺景深，夜晚冷色調，左側軟主光加暖色邊緣光分離主體與深色背景。不要文字，不要浮水印，不要其他路人。

🎯 適合場景：人物寫真、產品攝影、城市風光照

【模板二】中文文字圖像（海報/廣告）

海報 · 廣告 · 品牌
Prompt 結構
建立一張 [比例] 的 [資產類型]，主體是 [描述]。 畫面上方居中顯示：「[主要文字]」，字體 [字體描述]，顏色 [顏色]。 畫面下方居中顯示：「[副標文字]」，字體 [字體描述]，顏色 [顏色]。 [風格/色調/氛圍描述]。所有文字必須準確無誤。
範例提示詞（可直接使用）

建立一張 3:4 直向的課程宣傳海報，主體是一隻正在接受年青學員護理的可愛柴犬，背景是溫暖的室內環境。畫面上方居中顯示：「香港基督教女青年會」，字體為粗體無襯線字，顏色為品牌藍色。畫面中央顯示：「寵物護理員基礎證書課程」，字體為粗宋體，顏色為專業深灰。畫面下方居中顯示：「現正招生｜ERB課程」，字體為中等粗細，顏色為淺灰色。暖色調，活潑正面氛圍，乾淨專業的商業設計。所有文字必須準確無誤。

🎯 適合場景：課程宣傳、社群廣告、活動海報

【模板三】建築/風景寫實

建築 · 地景 · 旅遊
Prompt 結構
[具體建築/地點名稱] 的 [具體角度/視角] 照片， [拍攝時間/光線條件]，[相機與鏡頭設定]， [風格，如寫實/電影感]，[額外細節如天氣/季節]。 確認 [建築/地點] 的 [具體細節] 準確。
範例提示詞（可直接使用）

尖沙咀香港文化中心與幻彩詠香江夜景，標準視角從九龍一角拍攝，金黃色人造光源與城市倒影交織。寫實攝影，Canon EOS R5，24mm廣角鏡頭，三腳架長曝光，乾淨的構圖，無多餘人物。確保建築輪廓與實際外觀一致。

🎯 適合場景：旅遊素材、城市介紹、房地產宣傳

【模板四】角色一致性多圖生成

角色 · 連環 · 品牌
Prompt 結構
以這張圖片為角色參考，生成 [數量] 張不同場景的 [動作/表情]， 每張圖片中角色外貌必須與原圖完全一致， 包括 [具體特徵描述]。場景風格為 [風格]。
範例提示詞（可直接使用）

以這張圖片為角色參考，生成4張同一角色的不同場景：圖1在維多利亞港看夜景，圖2在茶餐廳吃早餐，圖3在太平山頂遠眺，圖4撐傘走在中環街道。每張圖中角色的黑色短髮、白皙皮膚、紅色外套必須與原圖完全一致。寫實攝影風格，電影感色調。

🎯 適合場景：故事漫畫、角色設定、行銷素材

【模板五】風格轉換（Style Transfer）

風格 · 藝術化 · 再利用
Prompt 結構
保持原圖的 [構圖/主體位置/視角] 完全不變， 將圖像轉換為 [目標風格]：[具體風格描述 + 色調 + 質感 + 光線]。 不要添加新元素，不要改變主體位置，不要添加文字。
範例提示詞（可直接使用）

保持原圖中人物與場景的構圖和位置完全不變，將圖像轉換為水彩插畫風格：柔和暈染邊緣、紙張質感、溫和的色彩暈開。顏色應更明亮但不過度飽和。不要添加新元素，不要移動主體，不要添加文字。

🎯 適合場景：品牌風格統一、藝術化加工、現有素材再利用

【模板六】商品乾淨背景圖

電商 · 產品 · 商業
Prompt 結構
[產品名稱與詳細描述]，位於 [位置/姿勢]， 白色或純色背景，[產品角度/展示方式]， 工作室級商業攝影光線（[光線描述]）， [風格/光線/色調要求]。不要背景裝飾物，不要浮水印，不要文字。
範例提示詞（可直接使用）

一個透明玻璃香氛蠟燭罐，啞光磨砂白色罐身，品牌名稱以金色燙金字體印於正面下半部，頂部有木質圓形蓋。產品置於畫面中央，45度斜側角度，淺灰漸層背景。工作室商業攝影，均勻柔光箱光源，精準還原玻璃通透感與木蓋質感。不要背景裝飾物，不要浮水印，不要文字。

🎯 適合場景：電子商務、產品型錄、客戶提案素材

【模板七】漫畫/分鏡創作

漫畫 · 敘事 · 社群
Prompt 結構
[數量] 格橫向漫畫分鏡，比例 [如 4:1]， 故事講述 [具體劇情描述]，風格為 [漫畫風格，如日式少年漫/法式漫畫]。 保持角色設計在所有格子中一致， 每格有不同的 [表情/動作/場景變化]。
範例提示詞（可直接使用）

4格橫向漫畫分鏡，比例 4:1，講述一個穿校服的高中生在茶餐廳點了一杯鴛鴦，卻發現杯子裡是菠蘿油的故事。使用日式少年漫畫風格，清晰的黑線稿配網點紙。角色穿白色襯衫、深藍色西裝褲、黑色皮鞋，保持所有格子一致。第四格加入爆炸效果框強調角色驚訝表情。

🎯 適合場景：社群內容、故事敘述、行銷短漫

【模板八】資訊圖表（Infographic）

資訊 · 教育 · 數據
Prompt 結構
建立一張 [比例] 的資訊圖表，主題是 [主題描述]， 包含 [數量] 個主要資訊區塊：[具體內容描述]。 使用 [色調，如暖色/科技感/清新] 配色方案， [字體/排版風格]，乾淨的版面設計。 [附加要求，如：包含標題/包含數據視覺化]。
範例提示詞（可直接使用）

建立一張 4:5 的繁體中文資訊圖表，主題是香港十大必吃傳統美食，包含5個主要資訊區塊：菠蘿油、蛋撻、奶茶、雲吞麵、咖哩魚蛋。使用傳統香港茶餐廳風格配色（米白、紅、綠、奶白），大字標題，副標題解釋，乾淨現代的版面設計，每個美食區配有插圖化呈現。

🎯 適合場景：教育內容、社交媒體分享、品牌資訊傳遞

【模板九】圖像局部編輯（加入元素）

合成 · 編輯 · 加工
Prompt 結構
在原圖的 [指定位置] 加入 [新元素詳細描述]， 新元素的光線方向必須與原圖現有光源一致， 透視角度必須與原圖匹配。 不要遮擋主體，不要改變整體色調與風格， 保持所有現有元素位置不變。
範例提示詞（可直接使用）

在原圖街道場景的右側加入一台經典紅色的士，停在路邊。紅色車身乾淨有光澤，金色車牌。車身受左側路燈暖光照射，投影方向與原圖其他車輛一致。透視角度為輕微仰視，符合街道消失點透視。不要遮擋主體人物，不要改變整體電影感色調，保持所有現有元素位置不變。

🎯 適合場景：場景豐富化、電影感加工、品牌元素植入

【模板十】UI/介面設計圖

UI · App · Mockup
Prompt 結構
建立一個 [設備] 上的 [應用類型] 介面設計截圖， 主畫面顯示 [功能模組描述]，整體採用 [設計語言]，使用 [主色調] 配色， 圓角 [大小]，陰影 [風格]。乾淨的高保真截圖，無狀態列干擾。
範例提示詞（可直接使用）

建立一個 iPhone 17 Pro 上的繁體中文天氣應用介面截圖，主畫面顯示當前天氣卡片（28度，多雲，屯門區）、未來6小時逐時預報橫向列表、九日天氣概覽。使用極簡現代設計語言，主色調為藍白漸變，卡片圓角16px，輕柔陰影。乾淨的高保真 Mockup 截圖，狀態列乾淨無干擾字樣，背景模擬真實室內場景。

🎯 適合場景：產品提案、介面展示、簡報素材
